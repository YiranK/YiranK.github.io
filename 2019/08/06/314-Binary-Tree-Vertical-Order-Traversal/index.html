314. Binary Tree Vertical Order Traversal
Given a binary tree, return the vertical order traversal of its nodes’ values. (ie, from top to bottom, column by column).
If two nodes are in the same row and column, the order should be from left to right.

Solution

Take-away

  1. Consider the col index relationship between parent and its left child, between parent and its right child. No more math relationship. BFS traverse the tree also since col index has relationship between parent and children.
  2. Rearrange the order of lists of TreeNode value in the last and dynamically update the range of col index, rather than update the col index of each TreeNode dynamically. Although the col index is in sequence, use an unordered_map rather than append list one by one.
